# Årshjul, arbejdsgange og processer v. Agnete (kl.9-10)

Formål: high-level indføring i OS2s årshjul, hvordan I arbejder og fungerer som forening. 


Spørgsmål:
Hvordan skal selvevalueringen (som jeg skal sidde med) passe ind i årshjulet i OS2?
Hvordan er belastningsgraden i sekretariatet henover tid? 
Fortæl gerne om OS2 håndbogen - hvad er målet med den? 
Hvad fungerer godt og skidt? 
Hvad ser man ind i af forandringer? 
Hvordan bliver OS2 mere skalerbar?
Hvordan styrer og styrker man processer i en decentraliseret forening?
